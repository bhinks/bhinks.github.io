Quake 2

I run a dedicated Quake 2 server at quake2.ninjabot.me (temporarily down) with a rotating collection of classic deathmatch maps. The server uses the Q2Pro engine for enhanced gameplay and stability.

Installing the Client

Windows / Linux / Mac

  1. Download and install Q2Pro - a modern Quake 2 client with improved performance and compatibility
  2. Point the client to your Quake 2 game data directory (or let it download the required files)
  3. Launch Q2Pro

Additional Resources

Looking for classic content like player models, skins, and maps? Check out the extensive archives at tastyspleen.net.

Connecting to the Server

Once you have Q2Pro installed:

  1. Open the console with the ~ key
  2. Type: connect quake2.ninjabot.me
  3. Press Enter

Or use the multiplayer browser to find "ninjabot" in the server list.

Server Info

  • Address: quake2.ninjabot.me
  • Game Mode: Deathmatch (rotating maps)
  • Future Plans: Additional mods, stat tracking, and team game bots
